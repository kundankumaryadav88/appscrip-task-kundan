
# Appscrip Task - Kundan

This is the solution for the Appscrip task, built using **Next.js** and styled with **normal CSS**.

## Features

- Server-side Rendering (SSR) with **Next.js**
- Custom styling using **normal CSS**
- Fully responsive layout
- Filterable product listing with sorting
- Accessible Accordion component

## Technologies Used

- **Next.js** (14.2.3)
- **React** (18)
- **Normal CSS** for styling

## Getting Started

### Prerequisites

- Node.js (Recommended version: 16 or above)
- npm (comes with Node.js)

### Installation

1. Clone the repository:

    ```bash
    git clone https://github.com/kundankumaryadav88/appscrip-task-kundan.git
    ```

2. Navigate to the project folder:

    ```bash
    cd appscrip-task-kundan
    ```

3. Install the dependencies:

    ```bash
    npm install
    ```

### Running the App

1. Start the development server:

    ```bash
    npm run dev
    ```

    The app will be available at `http://localhost:3000`.

2. To build the app for production:

    ```bash
    npm run build
    ```

3. To start the production server:

    ```bash
    npm run start
    ```
